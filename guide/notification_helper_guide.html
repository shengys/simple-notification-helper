Simple Notification Helper

Simple Notification helper adalah helper sederhana untuk framework Codeigniter yang dapat digunakan untuk keperluan notifikasi. Misalnya: untuk keperluan menampilkan pesan error/notifikasi saat user login. Untuk menampilkan pesan error/notifikasi ini simple notification helper memanfaatkan session flash data, sehingga pesan error/notifikasi hanya terlihat untuk request pertama dan akan hilang pada request selanjutnya.

Ada 3 tipe notifikasi: red, blue, green. Red untuk notifikasi yang sifatnya kegagalan (pesan error). Green untuk notifikasi yang sifatnya keberhasilan (pesan sukses). Blue untuk notifikasi yang sifatnya netral. Perbedaan diantara ketiganya adalah pada class css dari tag div yang membungkus pesan tersebut ketika pesan tersebut ditampilkan.

Prerequisite: Agar simple notification helper dapat digunakan, library session harus terlebih dahulu di-load.

Petunjuk penggunaan

1. Download dan letakkan file notification_helper.php di direktori ./system/application/helpers

2. Panggil simple notification helper setiap kali akan digunakan di controller. $this->load->helper('notification_helper');

ATAU

2. Setting autoload dengan mengubah file ./application/config/autoload.php sehingga notification helper bisa digunakan setiap saat. Contoh: $autoload['helper'] = array('url','notification_helper');

Mengeset data

set_red_notification($message="")

Contoh: set_red_notification("Maaf username atau password yang anda inputkan tidak sesuai.");

set_blue_notification($message="")

Contoh: set_blue_notification("Anda mempunyai 3 pesan baru.");

set_green_notification($message="")

Contoh: set_green_notification("Data telah berhasil diupdate.");

Menampilkan data

Fungsi-fungsi berikut ini bisa dipanggil di view untuk menampilkan notifikasi(jika notifikasi tersebut di-set.)

print_notification()

Jika ada satu atau lebih notifikasi telah diset, maka pesan notifikasi akan ditampilkan. Jika tidak maka tidak ada pesan yg ditampilkan. Contoh: < div id="form">
< ?php echo print_notification() ?>
...
< /div>

Jika misalnya sebelumnya telah diset green notification seperti ini: set_green_notification("Data telah berhasil diupdate.");

Maka html yg digenerate menjadi < div id="form">
< div id="notification"> < div class="green-notification">Data telah berhasil diupdate.< /div> < /div>
...
< /div>

Jika misalnya sebelumnya tidak ada notification yang di-set, maka html yg digenerate menjadi < div id="form">
...
< /div>

print_red_notification()

Hanya menampilkan red notification (jika telah diset). Contoh: < div id="form">
< ?php echo print_red_notification() ?>
...
< /div>

print_blue_notification()

Hanya menampilkan blue notification (jika telah diset). Contoh: < div id="form">
< ?php echo print_blue_notification() ?>
...
< /div>

print_green_notification()

Hanya menampilkan green notification (jika telah diset). Contoh: < div id="form">
< ?php echo print_green_notification() ?>
...
< /div>
